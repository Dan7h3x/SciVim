# NvimPy

The bleeding edge neovim config for python + lua development (other langs supported with null-ls)

---

Requirements:

- `neovim > 0.5`
- fd
- ripgrep
- pyright
- npm
- stylua
- `Nerd Fonts`

---

---

---

#### **_Installation_**

```unix
git clone https://github.com/Dan7h3x/NvimPy.git ~/.config/nvim && rm -rf ~/.config/nvim/.git && nvim +checkhealth
```
