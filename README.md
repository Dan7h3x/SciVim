# NvimPy

The bleeding edge neovim config for python + lua development (other langs supported with null-ls)

---

Requirements:

- `neovim > 0.5`
- fd
- ripgrep
- pyright
- npm
- stylua
- jq
- zathura (pdf viewer for latex)
- ipython (repl)
- `Nerd Fonts`
